# Operator Manual

Ai Lab Repo Intelligence Suite 18 is operated as a local, GitHub-first product.

## Daily Operation

1. Run tests before release.
2. Review generated reports.
3. Approve GitHub publication only when the quality report is acceptable.
4. Keep secrets in environment variables, never in the repository.

## Owner Controls

- Approve: publish to GitHub.
- Reject: keep artifacts local and record why.
- Delay: keep improving without publishing.
